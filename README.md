# mes-droits-travail.fr

Calculateurs gratuits pour le droit du travail français.

**Stack :** Next.js 14 · Tailwind CSS · TypeScript · Vercel

## Outils disponibles

- Calculateur de préavis (démission, licenciement, cadre/non-cadre)
- Simulateur rupture conventionnelle
- Calculateur indemnité de licenciement
- Convertisseur salaire brut / net
- Calculateur heures supplémentaires
- Simulateur chômage ARE
- Calculateur congés payés

## Démarrage

```bash
npm install
npm run dev
```

## Déploiement

Déployé sur [Vercel](https://vercel.com). Connecter le repo et configurer le domaine `mes-droits-travail.fr`.
